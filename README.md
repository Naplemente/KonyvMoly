# KonyvMoly
Webes alapú könyv kölcsönző rendszer, amely lehetővé teszi a könyvek, felhasználók és kölcsönzések kezelését. A projekt SQL adatbázist, JavaScript alapú frontend logikát, valamint HTML + CSS felhasználói felületet használ. Célja egy egyszerű, átlátható és jól bemutatható rendszer implementációja vizsgaremekként, csapatmunkában.
